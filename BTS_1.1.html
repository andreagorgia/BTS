A cosa serve bootstrap Bootstrap è un framework utilizzato per la programmazione front-end, il quale contiene template di design HTML e CSS per la creazione di vari tipi di tabelle, bottoni...
Cosa devi importare per creare una pagina web con bootstrap Bisogna importare i seguenti script:
<meta charset="utf-8">
<meta name="viewport" content="width=device-width, initial-scale=1">
<link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/4.3.1/css/bootstrap.min.css">
<script src="https://ajax.googleapis.com/ajax/libs/jquery/3.4.1/jquery.min.js"> <script src="https://cdnjs.cloudflare.com/ajax/libs/popper.js/1.14.7/umd/popper.min.js"> <script src="https://maxcdn.bootstrapcdn.com/bootstrap/4.3.1/js/bootstrap.min.js">
Cos’è un CDN? Che vantaggi comporta rispetto a scaricare bootstrap sul tuo sito? CDN(ContentDeliveryNetwork) è un sistema di server distribuiti in diverse aree geografiche che permette di inviare dei contenuti web ad un utente senza che lo stesso li abbia scaricati in locale, avendo tempi di risposta minori
A cosa serve questo attributo del tag meta?content="width=device-width, initial-scale=1 Serve a gestire la scalabilità della pagina in base al device in uso, tenendo conto pure dello "zoom".
